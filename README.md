Simple Autoclicker for adding timing to FSAE Injector Testing Protocol

# Controls
alt + (1, 2, 3, 4, 5, 6, or 0): Clicks, waits, clicks again
- 1: 10  seconds
- 2: 15  seconds
- 3: 20  seconds
- 4: 30  seconds
- 5: 60  seconds
- 6: 90  seconds
- 7: 120 seconds
- 0: Custom wait time


  Esc: (kills program)
